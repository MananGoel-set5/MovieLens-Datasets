# Movie Recommendation System Using Hybrid Learning

## Overview
This project builds a hybrid movie recommendation system using the MovieLens dataset. 
It combines unsupervised learning (K-Means clustering and Matrix Factorization) with 
supervised learning models (Gradient Boosting and Neural Networks).

## Dataset
MovieLens Dataset containing:
- user_id
- movie_id
- rating
- timestamp

## Methods Used

### Unsupervised Learning
- K-Means Clustering
- PCA Dimensionality Reduction
- Matrix Factorization (SVD)

### Supervised Learning
- Gradient Boosting Regressor
- Neural Network (MLP)

### Hybrid Model
Combines user clusters with rating prediction models.

## Evaluation Metrics
- RMSE (Root Mean Squared Error)
- Precision
- Recall

## Visualizations
- Correlation Matrix

- Model Comparison Graph
